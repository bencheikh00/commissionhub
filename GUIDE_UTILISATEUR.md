# 📱 Guide Utilisateur - CommissionHub

## 🚀 Démarrage rapide

### 1. Lancer l'application
```bash
npm run dev
```
Ouvrir http://localhost:4028

### 2. Se connecter
- Entrer votre email et mot de passe
- Ou créer un compte si vous êtes nouveau

### 3. Code secret
- Entrer le code : **BOURAMA**
- Accès au dashboard

## 🎯 Navigation

### Menu principal (☰)
Cliquer sur les **3 traits** en haut à gauche pour accéder à :

1. **👥 Membres** - Voir tous les membres de la commission
2. **📅 Demande d'absence** - Informer de votre absence
3. **🏆 Anciens présidents** - Historique des présidents
4. **💡 Boîte à idées** - Partager vos suggestions
5. **🎨 Galerie des logos** - Télécharger les logos officiels
6. **🚩 Signaler un problème** - Support technique

### Menu utilisateur
Cliquer sur votre **avatar** en haut à droite pour :
- ⚙️ Paramètres du profil
- 🚪 Déconnexion

### Mode sombre/clair
Cliquer sur l'icône **🌙/☀️** pour basculer entre les thèmes

## 📝 Utilisation des fonctionnalités

### Demander une absence
1. Ouvrir le menu (☰)
2. Cliquer sur "Demande d'absence"
3. Remplir le formulaire :
   - Date de début
   - Date de fin
   - Raison de l'absence
4. Cliquer sur "Envoyer la demande"

### Envoyer une idée
1. Ouvrir le menu (☰)
2. Cliquer sur "Boîte à idées"
3. Écrire votre suggestion
4. Cliquer sur "Envoyer l'idée"

### Télécharger un logo
1. Ouvrir le menu (☰)
2. Cliquer sur "Galerie des logos"
3. Choisir le logo souhaité
4. Cliquer sur "Télécharger"

### Signaler un problème
1. Ouvrir le menu (☰)
2. Cliquer sur "Signaler un problème"
3. Remplir le formulaire :
   - Sujet du problème
   - Description détaillée
4. Cliquer sur "Envoyer le rapport"

## 📱 Sur mobile

- **Menu** : Toucher les 3 traits (☰)
- **Retour** : Toucher la croix (✕) ou le bouton retour
- **Scroll** : Faire défiler pour voir tout le contenu
- **Formulaires** : Remplir directement sur la même page

## 💡 Astuces

- ✅ Tout se passe dans la même page, pas de redirection
- ✅ Le menu se ferme automatiquement après sélection
- ✅ Les formulaires affichent une confirmation après envoi
- ✅ Le mode sombre est sauvegardé automatiquement
- ✅ Les animations rendent l'interface fluide
